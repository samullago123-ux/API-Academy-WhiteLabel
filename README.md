[README (1).md](https://github.com/user-attachments/files/26726151/README.1.md)# 🚀 API Academy — Whitelabel

Plataforma interactiva para aprender APIs desde cero hasta nivel arquitecto. 3 niveles, 25 lecciones interactivas y 48 preguntas de quiz.

Construido con React + Vite.

---

## 📚 Niveles

### ⚡ Nivel 1 — API Basics
Fundamentos para quienes nunca han trabajado con APIs.

- Anatomía de un Request HTTP
- Métodos HTTP (GET, POST, PUT, DELETE)
- Códigos de Estado (200, 404, 500...)
- Headers y Autenticación
- JSON: el idioma de las APIs
- API Playground interactivo
- Quiz Final

### 🔥 Nivel 2 — Advanced APIs
Conceptos intermedios para trabajar con APIs en producción.

- OAuth 2.0 (Authorization Code, Client Credentials, PKCE)
- Rate Limiting y Exponential Backoff
- Idempotencia
- Paginación (Offset, Cursor, Keyset)
- Webhooks
- Versionamiento de APIs
- Errores y Resiliencia (Circuit Breaker)
- API Gateway
- Quiz Avanzado (20 preguntas aleatorias)

### 👑 Nivel 3 — Expert Design
Arquitectura y diseño de sistemas para nivel senior.

- API Design Patterns (HATEOAS, naming conventions)
- GraphQL vs REST
- Event-Driven Architecture
- Seguridad (OWASP API Top 10)
- Performance y Cache (Redis, CDN, ETags)
- OpenAPI y Documentación
- Sistemas Distribuidos (CAP, Sagas)
- Caso Real: diseñar una arquitectura completa
- Quiz Experto (20 preguntas aleatorias)
